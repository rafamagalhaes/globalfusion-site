    Cases

    Um case de sucesso foi na Inventcloud.

    Na inventcloud o ambiente de monitoramento estava sem padrão de nomenclatura de host e tags, o time de NOC ficava perdido. A integração entre o Zabbix e o GLPI era realizada por e-mail, o que causava atraso nas abertura de tickets e até mesmo falha desta. O histórico de monitoração possuía apenas 7 dias de retenção, impossibilitando qualquer estudo de capacity.

    Trabalhamos em conjunto com a equipe de NOC para entender as dificuldades e planejar as melhorias. O projeto durou 8 meses e englobou as seguintes atividades:

    - Padronização dos nomes de hosts e tags;
    - Padronização dos templates de monitoramento;
    - Integração do Zabbix com o GLPI via API do GLPI;
    - Criação de um repositório no GitLab para armazenamento dos códigos da equipe de monitoramento;
    - Criação de esteiras CI/CD para automações de processos e deployments;
    - Desenvolvimento de todo ambiente de monitoramento em microserviços;
    - Desenvolvimento de API para integração do OCI Monitoring com o Zabbix;
    - Melhoria do monitoramento de banco de dados Oracle via Zabbix Agent2;
    - Aumento da retenção do histórico para 1 ano e ajuste do servidor de banco de dados para suportar esse aumento;
    - Criação do HA do banco de dados com o InnoDB Cluster;
